## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/q-rico.git

  # Enter folder
  $ cd q-rico

  # Install dependences
  $ yarn

  #Execute
  $ yarn dev

```
